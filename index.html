02

正向能量：轉化為專注力

孩子的活力不該只是被壓抑，而是需要正確的出口。透過武術、拳擊與體能課程，引導孩子將爆發力轉化為控制力、反應力與行動力。

性格養成：課本外的生命教育

運動場也是最好的教室。在訓練中，孩子學習面對挑戰、尊重規則、配合夥伴，並在一次次完成任務的過程中，累積受用一生的自信心。

FAQ

常見問題

Q1：你們是安親班嗎？
+
不是。武樂是以兒童運動才藝課程為核心的運動空間，主要提供武術、拳擊、體能、翻滾等課程。是運動教室，不是傳統安親班或課後照顧中心。
Q2：孩子可以在武樂寫功課嗎？
+
可以安排安靜作業或閱讀時間，讓孩子在課程前後自主完成作業。但武樂主要服務為運動才藝課程，並非文理補習或作業輔導班，也不保證完成所有作業。
Q3：可以從學校接孩子過去嗎？
+
可以先討論。是否能協助接送，會依學校位置、放學時間、當期人數與路線安排評估。
Q4：可以從安親班接過去上課嗎？
+
可以討論。若安親班同意配合接送銜接，我們可依實際路線與時間評估是否安排。
Q5：小朋友沒有運動基礎可以參加嗎？
+
可以。課程會依孩子年齡、程度與身體狀況調整，初學者也能從基礎開始。
Q6：課程會不會太激烈？
+
不會一開始就安排高強度內容。兒童課程會以安全、趣味、循序漸進為原則，重點是建立身體能力、紀律與自信。
Q7：可以只固定某幾天去嗎？
+
可以。家長可以依孩子的補習班、才藝班或家庭安排，選擇固定某幾天參與武樂課程。
Q8：適合幾歲孩子？
+
主要適合國小階段孩子。實際是否適合會依孩子年齡、身體狀況與課程內容評估。
Q9：需要準備什麼？
+
建議穿著方便活動的衣服、運動鞋，並攜帶水壺、毛巾。若參加拳擊或武術進階課程，可能會依課程需要準備相關裝備。
Q10：可以先參觀嗎？
+
可以。歡迎家長提前預約參觀武樂運動空間，我們會協助說明課程安排與接送銜接方式。
